
## CreateCluster

**Provider: ROS**
create cluster use ros provider
```
 ./bin/ooc \
    create \
    --name ooc-stack-006 \
    --resource cluster \
    --provider ros \
    --provider-config-file /Users/aoxn/work/ooc/pkg/iaas/provider/ros/example/provider.cfg \
    --boot-config /Users/aoxn/work/ooc/pkg/iaas/provider/ros/example/bootcfg.yaml
```
